七牛云截屏上传返回图片地址

发表于 2019-08-22 分类于 markdown 阅读次数：
本文字数： 177 阅读时长 ≈ 1 分钟

  • 1.到github下载release版本

  • 2.打开release中的 settings.ini 文件

  • 3.去七牛云找到密匙填写

  • 4.BUCKET_NAME：储存空间的名称

  • 5BUCKET_DOMAIN：你自己的域名

  • 6.UP_HOST = http://up-z2.qiniup.com

  • 7.DEBUG_MODE：调试开关，截屏上传失败可以把false改为true
